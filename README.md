# Repository map (saralobo)

Index of all repositories by **category**. Each repo uses a prefix that identifies its type.

**Naming convention:**

| Prefix | Type |
|--------|------|
| `rules-` | Cursor rules (domain-specific instructions for AI agents) |
| `skill-` | Cursor/Claude skills (reusable AI workflows) |
| `agents-` | Agent systems (multi-agent packs, orchestrators, pipelines) |
| `mcp-` | MCP servers (Model Context Protocol) |
| `lib-` | Libraries (patterns, tokens, reusable code) |
| `proto-` | Prototypes (product experiments) |
| `archive-` | Archives (file storage, references) |

On GitHub, filter using **topics**: `topic:rules`, `topic:skill`, `topic:agents`, `topic:mcp`, `topic:lib`, `topic:prototype`, `topic:archive`.
